# Macbook Setup

Update to latest version of Sequoia.
Don't bother updating to Tahoe.

1. Install Homebrew (it will also install xcode tools)
1. Install: firefox, alacritty, btop, zoxide, fish, neovim, chezmoi, eza, fd, ripgrep, tldr, bat, delta, tmux, zellij, rclone, gimp, fdupes, aerospace, zola, tree
  - `brew install --cask firefox gimp` (graphical programs)
  - `brew install --cask nikitabobko/tap/aerospace`
  - `brew install --cask alacritty fish btop zoxide` (CLI programs)
  - `brew install --cask font-sauce-code-pro-nerd-font` (fonts)
1. Configure: alacritty, fish, neovim, bat
  - Install starship

1. Firefox plugins: Bitwarden, UBlock Origin, Vimium, Sidebery
1. Logins: Email (+ Calendar + Keep), Github, Amazon
1. Install: Signal, Slack, KKT
1. Configure aerospace
1. Clone repos: website, notes, research-notes, sim-notes, pubs, backup
1. Install VSCode
